Endpoints

Server : /

  • get /

Ingredients : /

  • /ingredient GET to retrieve all ingredients
  • /ingredient/id GET to retrieve an ingredient
  • /ingredient/ POST to add an ingredient
  • /ingredient/id PUT to edit an ingredient
  • /ingredient/id DEL to delete an ingredient

Recipies : /

  • /recipe GET to retrieve all recipies
  • /recipe/id GET to retrieve a recipe
  • /recipe/ POST to add a recipe
  • /recipe/id PUT to edit a recipe
  • /recipe/id DEL to delete a recipe
