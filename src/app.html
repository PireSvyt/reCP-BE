Endpoints

Server : /

  • GET

Ingredients : /ingredient

  • GET / to retrieve all ingredients
  • GET /id to retrieve an ingredient
  • POST / to add an ingredient
  • PUT /id to edit an ingredient
  • DEL /id to delete an ingredient

Recipies : /recipe

  • GET / to retrieve all recipies
  • GET /id to retrieve a recipe
  • POST / to add a recipe
  • PUT /id to edit a recipe
  • DEL /id to delete a recipe

Transactions : /recipe

  • GET / to retrieve all transactions
  • GET /id to retrieve a transaction
  • POST / to add a transaction
  • PUT /id to edit a transaction
  • DEL /id to delete a transaction
